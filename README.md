# How to Play
# Read the recipe on the board to the right
# Pick up the correct ingredient bottles and tilt them over a glass to pour
# Hold the tilt for about 1 second per ingredient
# Once all ingredients are poured, place the glass in the green zone on the counter
# The game will tell you if the drink is correct and move to the next recipe
# Recipes
# Drink	Ingredients
# Vodka Soda	Vodka, Soda Water, Ice
# Vodka Cranberry	Vodka, Cranberry Juice, Ice
# Martini	Gin, Vermouth
# Margarita	Tequila, Lime Juice, Triple Sec
